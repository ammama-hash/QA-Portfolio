## Tools Used
- **Selenium (Python):** Automated functional and UI test cases  
- **Apache JMeter:** Performance, load, and stress testing  
- **OWASP ZAP / Burp Suite:** Security vulnerability analysis  
- **GitHub Actions / Jenkins :** CI/CD pipeline integration  
- **Jira / Excel:** Bug reporting and tracking  

##  CI/CD Workflow Explanation
1. **Trigger:** Developer pushes code → pipeline starts automatically  
2. **Build Stage:** Dependencies installed, environment prepared  
3. **Test Stage:** Selenium scripts executed for login, search, and checkout  
4. **Report Stage:** JUnit/Allure reports generated and uploaded as artifacts  
5. **Deploy Stage:** Application deployed only if all tests pass
